# Logistics Rental Space (LRS)

## Overview

**Logistics Rental Space (LRS)** is a web-based platform that connects truck owners with individuals and small businesses that need cargo transportation services.

The platform allows truck owners to share their unused truck capacity by publishing available cargo space, while renters can find suitable transportation options and reserve space for their shipments.

LRS aims to improve truck utilization, reduce transportation costs, and minimize unnecessary trips by enabling efficient sharing of available cargo space.

---

## Problem Statement

Many trucks operate with unused cargo capacity, leading to:

- Inefficient use of transportation resources.
- Higher delivery costs.
- Increased number of trucks on roads.

LRS addresses this problem by allowing truck owners to rent their unused space to multiple renters, improving transportation efficiency.

---

## Key Features

### 🚚 Truck Space Sharing

LRS allows truck owners to:

- Publish available truck space.
- Specify truck details, cargo capacity, route, and rental price.
- Manage available space and rental requests.

### 📦 Multi-Renter Booking System

The platform supports:

- Multiple renters sharing the same truck.
- Managing available cargo capacity.
- Booking requests between truck owners and renters.

### 🔍 Search and Filtering

Users can find suitable transportation options based on:

- Pickup location.
- Destination.
- Cargo requirements.
- Truck type.
- Available space.

### ⭐ Rating-Based Recommendation System

LRS improves trust between users through a rating-based recommendation system.

The system:

- Allows users to rate truck owners after completed transactions.
- Calculates owner ratings based on user feedback.
- Recommends highly rated truck owners to users.
- Sends recommendations to users when a truck owner with a high rating posts a new advertisement.

### 🗺️ Shipment Tracking

LRS provides shipment tracking features to help users monitor their deliveries.

The system includes:

- Displaying shipment routes using Google Maps.
- Tracking shipment progress.
- Showing shipment status updates:
  - Pending
  - In Transit
  - Delivered
  - Rejected
